# **App Name**: NovaAdmin Dashboard Template

## Core Features:

- Dashboard Overview: Provides a comprehensive view of key metrics, including user statistics, sales performance, revenue trends, and order summaries.
- User Management: Enables administrators to manage user accounts, roles, and statuses efficiently, including user creation, editing, and deletion.
- Data Visualization: Utilizes interactive charts (line, bar, pie) powered by Chart.js to visualize data trends and insights effectively.
- Form Handling: Offers a variety of form elements with validation styles, including inputs, selects, checkboxes, and more, for seamless data input and management.
- UI Components: Includes a range of pre-designed UI components such as buttons, cards, alerts, badges and modals to allow for fast prototyping.
- Settings and Customization: Allows users to configure profile settings, change passwords, and toggle between light and dark themes for personalized user experience.
- Authentication: Includes UI for login, registration and password retrieval.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) to establish a professional and trustworthy visual identity.
- Background color: Light gray (#F9FAFB), nearly white, for a clean, uncluttered backdrop that ensures content clarity and focus.
- Accent color: Slightly more saturated and dark version of purple (#7066e0) used to subtly highlight primary elements.
- Body and headline font: 'Inter', a sans-serif font, chosen for its modern, neutral appearance and excellent readability.
- Fixed left sidebar with a width of 260px to maintain consistent navigation and access to key features.
- Top navbar with a height of 64px providing quick access to essential functions and user settings.
- Subtle transition effects for menu interactions, state changes, and content updates to enhance user experience without being distracting.
