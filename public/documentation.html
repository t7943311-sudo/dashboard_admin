NovaAdmin Documentation

Welcome! This guide provides everything you need to get started with the NovaAdmin template.

1. Introduction

NovaAdmin is a professional, frontend-only admin dashboard template built with modern web technologies. It is designed to be a clean, flexible, and powerful starting point for any web application, SaaS, or backend interface.

This template is built on top of Next.js, React, and Tailwind CSS, providing a highly efficient and modern development experience.

2. Getting Started

To run the project locally, you need to have Node.js and npm installed.

  1. Open your terminal and navigate to the project directory.
  2. Install dependencies: 
    npm install
  3. Start the development server: 
    npm run dev
  4. Open http://localhost:9002 in your browser. (The default port is 9002, but it may be different if that port is in use).

3. Folder Structure

The project uses the Next.js App Router for file-based routing and organization.

  • /app: Contains all the application routes.
    • /app/(dashboard): The main dashboard layout and pages.
    • /app/(auth): Authentication-related pages (Login, Register).
  • /components: Contains all reusable React components.
    • /components/layout: Core layout components like Header and Sidebar navigation.
    • /components/ui: ShadCN UI components.
    • /components/charts: Reusable chart components.
  • /lib: Contains utility functions (utils.ts) and mock data (mock-data.ts).
  • /public: Static assets like images and the documentation file.
  • /app/globals.css: The main CSS file for global styles and theme variables.
  • tailwind.config.ts: The configuration file for Tailwind CSS.

4. Customization

Changing the Logo

The logo is defined in src/components/layout/Logo.tsx. You can replace the default SVG icon or text with your own branding.

Changing Colors

All theme colors are defined as CSS variables in src/app/globals.css under the :root (for light mode) and .dark (for dark mode) selectors. You can modify these HSL values to match your brand's color palette.


{`:root {
    --primary: 243 75% 59%; /* Indigo */
    --background: 210 17% 98%; /* Light Gray */
    /* ... more colors */
}`}

Adding a New Page

  1. Create a new folder inside /app/(dashboard)/ (e.g., mynewpage).
  2. Inside it, create a file named page.tsx.
  3. Add your page content as a React component.
  4. To add it to the sidebar, open src/components/layout/SidebarNav.tsx and add a new link to the mainMenuItems array.
